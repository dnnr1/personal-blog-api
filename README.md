# Personal Blog API

A RESTful backend API for the **Personal Blog** project, built with **Node.js**, **Express**, and **TypeScript**.  
This API handles authentication, posts, file uploads, and database operations for the blog frontend.

Frontend repository: https://github.com/dnnr1/personal-blog

---

## Features

- JWT-based authentication
- User management (signup, login)
- CRUD operations for posts
- PostgreSQL database with migrations (Knex)
- File uploads with MinIO (S3-compatible storage)
- Request validation with Zod
- Tests with Jest & Supertest

---

## Tech Stack

- **Node.js**
- **Express 5**
- **TypeScript**
- **PostgreSQL**
- **Knex.js** (SQL query builder & migrations)
- **UUID** (unique identifiers)
- **JWT (jsonwebtoken)**
- **bcrypt** (password hashing)
- **CORS** (cross-origin support)
- **MinIO** (object storage)
- **Multer** (file uploads)
- **Zod**
- **Lodash**
- **Jest**
- **Supertest**
